# Kestra Workflows

## smart-research-assistant.yaml

AI-powered research assistant that:

- Uses Kestra AI Agent to decide which databases to query
- Searches ArXiv for CS/Physics/Math papers
- Searches Semantic Scholar for interdisciplinary research
- Makes autonomous decisions based on research topic
