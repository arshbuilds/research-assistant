# Universal Research Assistant

AI-powered research assistant that searches academic papers and code repositories, summarizes findings, and makes intelligent decisions about which sources to explore.

## Tech Stack

- **Kestra**: Workflow orchestration with AI agents
- **Oumi**: Open-source LLM inference (Llama/Mistral)
- **Cline CLI**: Development workflow assistance
- **Vercel**: Deployment

## Data Sources

- ArXiv (CS/Physics/Math papers)
- Semantic Scholar (Cross-disciplinary academic papers)
- GitHub (Code implementations)
